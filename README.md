# Week-2
missing-data-pandas-practice
Missing Data Handling in Pandas – Practice Exercises
Overview
This repository contains practical exercises focused on detecting and handling missing data using Python and Pandas. The work follows structured guidance from the GeeksforGeeks tutorial on working with missing data in Pandas.
Dataset
Employees.csv
The dataset was used to practice identifying missing values and applying appropriate imputation techniques.
Objectives
The main objectives of this exercise were:
Detect missing values using:
isnull()
notnull()
Quantify missing data across columns
Replace missing values using:
fillna()
Ensure the dataset remains complete and ready for further analysis
Key Concepts Practiced
Identifying NaN values in a DataFrame
Filtering valid and invalid records
Replacing missing values without dropping useful data
Preparing datasets for preprocessing and modelling
Files Included
Week1_MissingData_Employees.ipynb – Jupyter Notebook containing working code and outputs
Employees.csv – Dataset used for the exercise
